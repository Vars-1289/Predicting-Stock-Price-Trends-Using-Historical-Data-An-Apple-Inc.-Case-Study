# Predicting-Stock-Price-Trends-Using-Historical-Data-An-Apple-Inc.-Case-Study
Predicting Apple (AAPL) next-day closing prices using historical data and technical indicators (RSI, MACD, Bollinger Bands, MAs). Compared Linear Regression, Random Forest, and LSTM models. Linear Regression achieved best performance (R²≈0.98). Includes real-time prediction support.
